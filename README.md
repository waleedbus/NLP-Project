# AI-Powered Document Classification System

This project is a simple NLP + Machine Learning application that automatically classifies administrative documents into categories such as:
- Report
- Form
- Request
- Email

# Features
- Text preprocessing and feature extraction using TF-IDF
- Document classification using Multinomial Naive Bayes
- Model evaluation with accuracy and classification report
- Saves trained model as a reusable `.joblib` file
- Predicts categories for unseen document text

# Tech Stack
- Python
- Pandas
- Scikit-learn
- Joblib
